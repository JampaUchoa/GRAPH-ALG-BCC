## Build

To build in a UNIX command line use 
```
gcc main.c GRAPH_LST.c GRAPH_MTX.c -o a.out
```

And to run

```
./a.out    
```

One off command:

```
gcc main.c GRAPH_LST.c GRAPH_MTX.c -o a.out && ./a.out && rm ./a.out
```
